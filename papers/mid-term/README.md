# MACR Mid-term Paper

A comprehensive, structured academic paper writing project for the MACR (Multi-Agent Collaborative Repair) mid-term report.

## Features

- **Markdown + YAML metadata**: Paper content lives in `sections/*.md`; metadata (title, authors, keywords) is centralized in `metadata.yaml`.
- **BibTeX references**: Literature managed in `references.bib`.
- **CSL citation styles**: IEEE, ACM, and ACL CSL files in `csl/`.
- **LaTeX conference templates**: IEEEtran, ACM acmart, and ACL styles in `templates/`.
- **Self-contained class files**: Required LaTeX classes (`IEEEtran.cls`, `acmart.cls`, `acl.sty`) are bundled under `templates/latex-classes/` so the project builds without additional global TeX Live packages.
- **Python figure pipeline**: Matplotlib-based figures plus a PGF/TikZ example, generated by `figures/generate_all.py`.
- **One-click build**: GNU `Makefile` with `make all`, `make ieee`, `make acm`, `make acl`, `make report`, `make figures`, `make clean`, and `make watch`.
- **Chinese mid-term report**: Separate `report.md` + `report_metadata.yaml` + `templates/report/` to produce `build/mid-term-report.pdf`.
- **Pandoc + XeLaTeX**: PDF engine is `xelatex`; build is driven by `pandoc` with Lua filters for includes and table conversion.

## Quick Start

```bash
cd papers/mid-term
make all
```

This generates:

- `build/mid-term-ieee.pdf`
- `build/mid-term-acm.pdf`
- `build/mid-term-acl.pdf`
- `build/mid-term-report.pdf`

## Build Targets

| Target | Description |
|--------|-------------|
| `make all` | Generate figures and build all four PDFs |
| `make figures` | Run all Python figure-generation scripts |
| `make ieee` | Build IEEE conference format PDF |
| `make acm` | Build ACM conference format PDF |
| `make acl` | Build ACL conference format PDF |
| `make report` | Build Chinese mid-term report PDF |
| `make clean` | Remove build artifacts and generated figures |
| `make watch` | Rebuild automatically on file changes (requires `entr`) |
| `make help` | Show available targets |

## Editing the Paper

1. **Metadata**: Edit `metadata.yaml` (title, authors, abstract, keywords).
2. **Content**: Edit files under `sections/`.
3. **References**: Add BibTeX entries to `references.bib`.
4. **Figures**: Add or modify scripts in `figures/` and register them in `figures/generate_all.py`.
5. **Cross-references**: This project does not use `pandoc-crossref`. Use explicit references (e.g., "Figure 1", "Table 1", "Section 3") or raw LaTeX `\label`/`\ref` if needed.

## Adding a New Citation Style

1. Download the desired `.csl` file into `csl/`.
2. Add a new template directory under `templates/` (or reuse an existing template).
3. Add a new target in `Makefile` mirroring `ieee`/`acm`/`acl`.

## Notes on LaTeX Classes

The project bundles conference class files so it builds out of the box:

- `IEEEtran.cls` from CTAN
- `acmart.cls` compiled from the official acmart DTX sources
- `acl.sty` and `acl_natbib.bst` from the ACL style files repository

If you prefer to use your TeX Live installation's copies, remove or empty `templates/latex-classes/` and the build will fall back to the system path.

## Dependencies

- `pandoc` (>= 3.0 recommended)
- `xelatex` (TeX Live 2023 or later)
- `make`
- `python3` with `matplotlib`

Optional for `make watch`:

- `entr` (available in most package managers as `entr`)

All Python dependencies are already included in the parent MACR virtual environment.
